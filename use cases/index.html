ID SignUpAccount
Назва: Створити обліковий запис
Учасники: Користувач, система
Передумови: - Користувач не зареєстрований
Результат: Новий обліковий запис
Виключні ситуації: - SignUpAccount_InvalidData_EXC - користувач ввів некоректні дані
- SignUpAccount_AlreadyExists_EXC - користувач ввів дані про обліковий запис, який вже існує
- SignUpAccount_CancelButton_EXC - користувач натиснув кнопку "Відміна"
ID LogIntoAccount
Назва: Увійти в обліковий запис
Учасники: Користувач, система
Передумови: - Користувач зареєстрований
- Користувач не авторизований
Результат: Авторизація у системі
Виключні ситуації: - LogIntoAccount_UnknownAccount_EXC - користувач ввів дані про неіснуючий обліковий запис
- LogIntoAccount_WrongPassword_EXC - користувач ввів неправильний пароль
- LogIntoAccount_CancelButton_EXC - користувач натиснув кнопку "Відміна"
